# Rocket Codebase Source Code

This repository is a monorepo containing the full software stack for the rocket program and its supporting systems.
It includes the main flight firmware, radio module firmware, and ground station firmware, with all three implemented as modular systems to support maintainability and scalable development.
The codebase also contains the obc_app project, which provides the operational on-board control application components.
In addition, it includes the telemetry project for communication and data handling across mission workflows.
Simulation environments and supporting tools are provided to enable testing, validation, and system-level integration before deployment.
Overall, this is a comprehensive, enterprise-grade codebase designed to support robust engineering, collaboration, and long-term evolution.
